## 3. 4 축 통합 보고 절차

### 단계 1 — 4 축 영역 영구 참조

전체-현황-가시화-표준.md SSoT §2 인용:

| 축 | 참조 영역 |
|----|---------|
| 축 1 — PRJ | `프로젝트/_dashboard.md` |
| 축 2 — 작업 항목 | `프로젝트/PRJ-2026-014_*/05_산출물/20260512-작업항목-INDEX.md` §4·§5·§6 |
| 축 3 — 학습 | `규칙/프로세스/자비스-운영-룰.md` §2 |
| 축 4 — 사이클 | 인덱스 §11 변경 이력 |

### 단계 2 — 통합 보고 본문 골격

```
## 전체 현황 가시화 보고 (4 축 통합)

**조회 시점**: {시점 분기 — 세션 시작·종착·사이클 진입·종착}

### 축 1 — PRJ 영역
- 진행 중 PRJ: N건 (리스크 M건)
- 최근 진행 일자

### 축 2 — 작업 항목 영역
- 28 작업 항목 영역 완수율: X%
- 11 게이트 완수율: X%

### 축 3 — 학습 영역
- 누적 학습: #1~#34 (#12~#19 예약)
- 최근 신설: #N

### 축 4 — 사이클 영역
- W-N 사이클 진행률: X%
- 최근 사이클 종착 일자
```

### 단계 3 — 다음 우선순위 영역 안내

- 미결 영역 + 우선순위 보고
- 다음 사이클 진입 권고 영역
