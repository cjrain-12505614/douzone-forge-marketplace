---

Claude Code에서 개발할 사항을 확인하는 커맨드다. 아래 순서대로 수행하라.

1. `CLAUDE.md`의 "역할 분담" 섹션에서 **현재 활성 스프린트** 번호를 확인한다.
2. 해당 스프린트의 구현 지시서를 읽는다:
   - 경로: `docs/02_기획/요구사항/sprint{N}_claude_code_instructions.md`
3. 지시서 내용을 기반으로 아래 형식으로 보고한다:

```
## 현재 스프린트: Sprint {N}

### 전체 Step 현황
| Step | 항목 | SP | 상태 |
|------|------|-----|------|
| 0 | ... | - | ✅/⬜ |
| 1 | ... | 5 | ✅/⬜ |
| ... | | | |

### 다음 실행할 Step
**Step {X}: {제목}** ({SP} SP)

{지시서에서 해당 Step의 지시 내용 요약}

### 시작하려면
"Step {X} 진행해" 라고 말씀해주세요.
```

4. 이미 완료된 Step이 있는지 확인하려면:
   - `docs/구현보고서/` 폴더에서 해당 스프린트의 구현 보고서가 있는지 확인한다.
   - Git 로그에서 관련 커밋이 있는지 확인한다.
   - 완료 여부가 불확실하면 빌드(`./gradlew clean build`)를 실행하여 코드 상태를 점검한다.
