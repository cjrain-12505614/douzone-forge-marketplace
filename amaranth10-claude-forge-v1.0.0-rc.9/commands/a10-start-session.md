name: a10-start-session
description: "세션 시작 프로토콜 실행"
---

세션 시작 프로토콜을 실행한다. 아래 순서대로 수행하라.

1. `CLAUDE.md`를 읽어 프로젝트 지침을 로드한다.
2. `projects/_dashboard.md`를 읽어 전체 프로젝트 포트폴리오 현황을 파악한다.
3. 사용자가 특정 모듈을 지정했으면 해당 모듈, 아니면 최근 작업 모듈의 `_개인/sessions/{모듈}/_current.md`를 확인한다.
   - 존재하면 → 이전 세션 진행 상황을 읽고 남은 작업을 파악한다.
   - 없으면 → 새 세션으로 시작한다.
4. 해당 모듈의 `tasks/_current.md`를 읽어 현재 업무 목록을 확인한다.
5. PM에게 아래 형식으로 보고한다:

```
PM에게 보고합니다.

**이전 세션**: (날짜, 주요 완료 사항 1~2줄 — sessions/_current.md 기반)
**진행 중 프로젝트**: (PRJ-YYYY-NNN 목록 — _dashboard.md 기반)
**현재 업무**: (tasks/_current.md 기반 주요 항목)
**이번 세션 목표 제안**: (이전 세션 남은 작업 또는 tasks 기반 1~3건)

이 방향으로 진행할까요?
```
