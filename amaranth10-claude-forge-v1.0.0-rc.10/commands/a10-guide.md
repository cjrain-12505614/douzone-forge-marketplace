# /guide 커맨드

프로젝트의 현재 상태를 파악하고, 다음에 무엇을 해야 하는지 안내합니다.

## 실행 절차

1. **현재 상태 진단**
   - CLAUDE.md 존재 여부 확인
     → 없으면: `/init-project`부터 하라고 안내
   - CLAUDE.md의 로드맵(Section 7) 읽기 → 현재 Phase 파악
   - 최신 세션 로그 읽기 → 마지막 작업과 다음 TODO 확인
   - 미결 의사결정(Section 9) 확인 → 대기 중인 결정이 있으면 우선 안내

2. **상태별 안내**

   **📋 초기 (로드맵 비어있음)**
   - "프로젝트가 막 시작되었습니다. 추천 순서:"
   - 프로젝트 규모별 추천 경로 제시 (소/중/대)

   **🔄 진행중 (활성 Phase 있음)**
   - 현재 Phase와 완료율 표시
   - 미완료 항목 중 가장 우선순위 높은 것 안내
   - 미결 의사결정이 있으면 먼저 해결 권고

   **⏸ 중단됨 (마지막 세션 로그가 오래됨)**
   - "마지막 작업이 N일 전입니다. 이어서 하시겠습니까?"
   - `/start-session`으로 복귀 안내
   - 마지막 세션의 다음 TODO 표시

   **✅ 스프린트 진행중**
   - 현재 스프린트의 Step 진행률 표시
   - 다음 Step 안내
   - 검증 필요한 산출물 표시

3. **사용 가능한 커맨드 추천**
   - 현재 상태에 맞는 커맨드 2~3개만 추천 (전체 목록은 뿌리지 않음)
