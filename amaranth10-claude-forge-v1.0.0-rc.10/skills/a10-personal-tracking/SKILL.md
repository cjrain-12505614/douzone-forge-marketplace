# 개인 트래킹 (a10-personal-tracking) — Phase Q-2 Q-14 신설

본인이 부하 직원 트래킹 노트를 작성하는 스킬. **PT-02 G9 a10-team-tracking (폐기) → G10 a10-personal-tracking 폐기 후 재설계** (D6 결정).

## G9 → G10 차이 (재설계 핵심)

| 항목 | G9 a10-team-tracking (폐기) → G10 a10-personal-tracking (폐기) | G10 a10-personal-tracking (본 스킬) |
|------|---------------------------|------------------------------------|
| 위치 | `team-tracking/` 공용 영역 | `_개인/팀트래킹/` 본인 영역 |
| 권한 | 공용 — Cell 리더 → Unit장 → 본부장 계층 공유 | 개인 — 본인 작성·열람만 (계층 공유는 본인 의지) |
| 대상 식별 | 정태 6 관리자 폴더 (Step 7) | xlsx 부서 계층 동적 분석 — "하위 멤버 + 상위 직책자" |
| 일반화 | SBUnit 한정 표현 다수 | SBUnit 한정 0 (Phase Q-1 Q-02 일관) |

## 핵심 정책

- **본인 영역** — `_개인/팀트래킹/`은 GitLab 동기 안 함 (`.gitignore` 등록)
- **직급 매칭** — `a10-people-index` 호출 + xlsx 부서 계층 분석으로 본인 부하 자동 식별
  - "하위 멤버 존재" — 본인 부서에 직급 낮은 인원 존재
  - "상위 직책자" — 본인이 Cell 리더·Unit장·본부장 또는 그에 준하는 관리 책임
- **민감 정보** — 트래킹 노트는 평가·코칭 메모 포함 가능, 외부 공유 금지

## 입력

- **대상자 ID** (xlsx 13번 컬럼 또는 이름)
- **일자** (예: `20260429`, 미지정 시 오늘)
- **주제** (예: `주간트래킹`·`1on1`·`월간평가`·`프로젝트리뷰`)

## 실행 흐름

1. `a10-people-index` 호출 → 본인 ID 매핑 + 부하 식별
2. 입력 대상자 ID 매칭 → 이름·직급·소속 추출
3. 본인이 대상자 관리 권한 보유 검증 (직급 매칭)
4. 노트 폴더 신설 — `_개인/팀트래킹/{대상자_ID}/`
5. 노트 파일 생성 — `{YYYYMMDD}-{주제}.md`
6. frontmatter 자동 채움:
   ```yaml
   ---
   대상자_ID: {ID}
   대상자: {이름}
   소속: {Cell}
   직급: {직급}
   일자: {YYYY-MM-DD}
   주제: {주제}
   작성자_ID: {본인 ID}
   ---
   ```
7. 본문 5섹션 골격:
   ```markdown
   ## 사실 (Facts)
   - 진행 중 작업: ...
   - 완료된 산출: ...

   ## 평가 (Assessment)
   - 성과: ...
   - 부담/이슈: ...

   ## 액션 (Actions)
   - 즉시: ...
   - 1주 내: ...

   ## 다음 점검 일자
   - YYYY-MM-DD
   ```

## 출력 검증

```
✅ 트래킹 노트 신설
  - 대상: {이름} ({ID})
  - 위치: _개인/팀트래킹/{ID}/{YYYYMMDD-주제}.md
  - 권한: 본인 영역 (GitLab 미동기)
```

## 사용 사례

### 1. Cell 리더 — 본인 Cell 멤버 주간 트래킹
```
/a10-personal-tracking B6276 20260429 주간트래킹
→ _개인/팀트래킹/B6276/20260429-주간트래킹.md (손예진)
```

### 2. Unit장 — 4 Cell 리더 1:1
```
/a10-personal-tracking B3505 20260430 1on1
→ _개인/팀트래킹/B3505/20260430-1on1.md (유지수)
```

### 3. 본부장 — Unit장 분기 평가
```
/a10-personal-tracking B2772 20260331 분기평가
→ _개인/팀트래킹/B2772/20260331-분기평가.md (차민수)
```
