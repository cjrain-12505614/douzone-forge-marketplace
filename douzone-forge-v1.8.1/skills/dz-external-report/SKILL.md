# dz-external-report — 외부 보고 영역 본문 자동 작성·검증

## 트리거 영역

다음 어휘 영역 트리거 시 본 스킬 자동 활성:
- "외부 보고 작성"
- "사내 보고 본문"
- "ONEFFICE 보고"
- "보고서 작성"
- "메일 본문 작성"
- "사내 매핑 본문"
- "외부 영역 작성"

## 본 스킬 영역

본 스킬은 본 워크스페이스 사용자가 외부 영역(사내 ONEFFICE·메일·외부 회의 자료)에 직접 작성하는 본문 영역 자동 작성·검증 영역. 워크스페이스 내부 코드 자동 회피 + 일반 용어 매핑 매트릭스 적용 영역.

**의존 SSoT**: `규칙/프로세스/외부-보고-작성-표준.md` (PRJ-2026-014 외부 보고 작성 표준 정착 사이클 산출)

## 본 스킬 적용 절차 (4 단계)

| 단계 | 본문 |
|------|------|
| 1. 워크스페이스 본문 정합 영역 | 사용자 요청 영역 본문 + 본 워크스페이스 내부 본문(계획서·결과서·인덱스 등) 핵심 영역 정합 |
| 2. 일반 용어 변환 영역 | SSoT §2 회피 매트릭스 8 영역 적용 — 코드 일반 용어 매핑 |
| 3. 자가 점검 영역 | SSoT §4 5단계 자가 점검 적용 (학습 #28 4단계 + 코드 검출 1단계) |
| 4. 산출 영역 | 외부 영역 발행 영역 권장 형식 일관 (사내 ONEFFICE 영역 또는 메일 영역) |

## 산출물 권장 형식

```
## [외부 영역 본문 제목]

### 본문 요약
{핵심 본문 3~5줄 — 일반 용어 일관}

### 본문 상세 (옵션)
{필요 시 본문 영역}

### 진행 영역
- 종착 영역: {일반 용어 일관}
- 진행 영역: {일반 용어 일관}
- 다음 영역: {일반 용어 일관}
```
